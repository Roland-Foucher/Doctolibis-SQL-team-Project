# SQL projet group

## Contexte

Ce projet est un exercice effectué lors de notre formation développeur web et web mobile.
Le but est de travailler sur :

* La conception de projet en équipe
* l'étude de projet,
* les bases de données MySQL
* Composants d'accès aux données Java et JDBC (repository et entités)

Ce projet est une simulation d'application de prise de rendez-vous en ligne médicale (type Doctolib)

## Stack Technique

* Java
* JDBC
* MariaDB
* JUnit
